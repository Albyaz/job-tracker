# Job Tracker CLI

A command-line job application tracker built with Python and SQLite.

## Features
- Add job applications with company, role, location, salary, and notes
- View all applications in a clean, colour-coded table
- Update application status (Applied, Interview, Offer, Rejected)
- Delete applications
- Data stored locally in SQLite database

## Tech Stack
- Python 3.11
- SQLite3
- Rich (terminal formatting)

## How to Run
```bash
git clone https://github.com/Albyaz/job-tracker.git
cd job-tracker
python -m venv venv
source venv/bin/activate
pip install rich
python tracker.py
```
